        • MockSlingHttpServletResponse build(Closure closure)

          Build a Sling response using a closure to set response properties. The closure delegates to this builder and an instance of MockHttpServletResponse, so methods on the response instance may be called directly in the closure (see example below). This pattern is similar to the Groovy with method. 

            new ResponseBuilder().build {
      status = 200
      characterEncoding = "UTF-8"
      contentType = "application/json"
      addHeader "Connection", "close"
 }
          Returns:
          response
          Parameters:
          closure - closure that delegates to this builder and MockHttpServletResponse
