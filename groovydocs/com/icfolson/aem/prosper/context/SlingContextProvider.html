      • void registerAdapterFactory(AdapterFactory adapterFactory)

        Register an adapter factory for the current Prosper context. Use this method only if the adapter factory has an OSGi metadata XML file in the classpath at /OSGI-INF containing the adaptable and adapter classes. Otherwise, use the other permutation of this method to explicitly specify the required metadata values.

        Parameters:
        adapterFactory - adapter factory instance
